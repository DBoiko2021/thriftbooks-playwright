# ThriftBooks Playwright POM

Minimal end to end test of a ThriftBooks works page using Playwright with a small Page Object Model.

The flow
1) Open a specific Works page
2) Add the book to the Cart
3) Go to Cart & Checkout
4) Verify that the Cart & Checkout contains the same title as on the Works page

## Tech stack

Playwright Test, Node.js, JavaScript, Page Object Model
